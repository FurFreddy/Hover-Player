I created this small userscript for e621, e926 & e6AI because I was annoyed that GIFs and videos weren't playing when I scrolled through e6. The script simply skips the extra step of opening the posts. If you hover over any GIF or video, it will play in the thumbnail box.

To install use this link: 
https://raw.githubusercontent.com/FurFreddy/Hover-Player/refs/heads/main/Hover%20Player.user.js
